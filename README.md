# ste

<!-- badges: start -->
<!-- badges: end -->

The "ste" package equips users with the simulation tools and inference methods for the spectral transfer entropy (STE) framework.

## Installation

You can install the current version of the "ste" package from
[GitHub](https://github.com/) with:

``` r
# library(devtools)
devtools::install_github("ptredondo/ste")
```

## Reference

  Redondo, Paolo Victor, Raphaël Huser, and Hernando Ombao. (2025). "Measuring Information Transfer Between Nodes in a Brain Network Through Spectral Transfer Entropy" *The Annals of Applied Statistics, 19(3)*, 2386--2411.
